Help

General Usage

  • Enter in HLA-B typings to classify their leader alleles (M or T).
  • Can import a tabular file or manually input the typing information.
  • Sort once the patient and donor HLA-B typing information is complete.
  • Export the resulting information

Keyboard Shortcuts

  • Use tab to move on to the next HLA-B allele input field.
  • Use tab to submit input text as is.
  • Use shift + tab to go back to a previous input field.
  • To go through autocomplete suggestions, use ↑ or ↓. Press enter/tab to select.
